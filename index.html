History

Where does the name Arduino come from?

Contrary to popular naming conventions, the name of the Arduino doesn't come from the name of its creator. It in fact comes from a small bar in Ivrea, Italy where the creators would gather to share ideas for creating the Arduino microcontroller. The name of the bar? You guessed it, Arduino.

What is the mission of the creators of Arduino?

The Arduino project began in 2005 as a tool for students at the Interaction Design Institute Ivrea, Italy, aiming to provide a low-cost and easy way for novices and professionals to create devices that interact with their environment using sensors and actuators. Common examples of such devices intended for beginner hobbyists include simple robots, thermostats and motion detectors. Source, Wikipedia.

Hardware

The Board (UNO R3)

The board itself has a footprint similar to that of a deck of cards and only weighs 25g. It is based on the ATMega328P, a reliable microcontroller chip which is used to control the input and output pins on the board.

I/O

14 digital input/output pins (of which 6 can be used as PWM outputs), 6 analog inputs, a 16 MHz ceramic resonator, a USB connection, a power jack, an ICSP header and a reset button

IDE

Arduino Software

The board is programmed using either its own lightweight IDE or a tool from the command line. The Arduino language is based on C and C++ and utilizes a lot of the keywords from these languages. Upon opening a new "sketch" in the IDE the user is welcomed by a couple of methods

void setup();

and

void loop();

    /*
                        Thermometer based on 1n4148 silicon diode used as temperature sensor.The thermometer is based on the diode characteristic that the increase of the temperature its forward voltage (VF) is lowered by 2,2mV / Â° C.
                      Fixing the value of Vf = VF0 at ambient temperature t0, the temperature value
                      t is calculated with the following formula:
                      
                      t= t0 - [vf(t)- vf0]* K
                      
                      with K = 1 / 2,2mV
                      
                      The value of Vf (t) = dtemp -vf0 is obtained by averaging values of 1024 by acquiring as many vf values
                      
                      The result of t is shown on the serial monitor
                      */
                      // set pin numbers:
                      
                      const int in = A0;          // used to bias the diode  anode
                      const int t0 = 20.3;
                      const float vf0 = 573.44;
                      // variables will change:
                      
                      int i;
                      float dtemp, dtemp_avg, t;
                      
                      void setup() {
                        Serial.begin(9600);
                        pinMode(in, INPUT_PULLUP);            // set the  pin IN with npull up to bias the diode
                      }
                      
                      void loop() {
                        dtemp_avg = 0;
                        for (i = 0; i < 1024; i++) {
                          float vf = analogRead(A0) * (4976.30 / 1023.000);
                          //Serial.println(vf);
                          dtemp = (vf - vf0) * 0.4545454;
                          dtemp_avg = dtemp_avg + dtemp;
                        }
                        t = t0 - dtemp_avg / 1024;
                        Serial.print("temperature in Celcius degree)   = " );
                        Serial.println(t);
                        
                        delay (1000);
                      }
